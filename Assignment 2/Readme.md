# 📊 NLP Feature Extraction & Word Embeddings

## 📌 Overview

This project focuses on feature extraction techniques in NLP, including Bag of Words (BoW), TF-IDF, and Word2Vec embeddings using a real-world dataset.

## 📂 Dataset

Dataset used: Car Dataset from Kaggle
https://www.kaggle.com/datasets/CooperUnion/cardataset

## ⚙️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Gensim
* NLTK

## 🔍 Features

* Bag of Words (BoW)

  * Count-based word occurrence
  * Normalized count occurrence
* TF-IDF (Term Frequency - Inverse Document Frequency)
* Word2Vec embeddings for semantic representation

## 🚀 How to Run

1. Install required libraries:

   ```
   pip install pandas scikit-learn gensim nltk
   ```
2. Download the dataset and place it in the project directory
3. Run the Python script:

   ```
   python filename.py
   ```

## 📊 Output

* Word frequency matrix (BoW)
* Normalized frequency values
* TF-IDF vectors
* Word embeddings using Word2Vec
* Similar word predictions

## 🧠 Key Concepts

* Bag of Words: Counts word occurrences without context
* Normalized BoW: Converts counts into relative frequency
* TF-IDF: Weighs words based on importance
* Word2Vec: Generates vector representations capturing semantic meaning

## 📌 Example Use Case

* Text classification
